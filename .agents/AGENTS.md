# Versioning Rule

Whenever you (the AI) fulfill a request and modify code in this project, you must automatically increment the version number in `src/app/layout.js`.
The version number is located in a `<span>` tag below the "NOC Control Center" title (e.g., `v1.0.1`).
Increment the patch version (e.g., `1.0.1` -> `1.0.2`) as part of your changes before finalizing your response to the user.
